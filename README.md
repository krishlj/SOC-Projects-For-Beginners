Splunk SIEM Log Analysis Projects

Repository Overview

This repository contains practical projects focused on investigating different types of log data using Splunk SIEM. The projects are designed for beginner and intermediate defensive security professionals who want to develop hands-on skills in log analysis, threat detection, security monitoring, and incident investigation.

Each project follows a structured process for importing sample logs into Splunk, identifying important fields, running SPL searches, detecting suspicious activity, and reviewing the results from a SOC analyst’s perspective.
